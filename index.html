Meal Tracker Application

A web app that keeps track of the users meals. A user can sign in and input the type of meal they ate and give a description on what they ate and other info they like to include. Future features would include using the Department of Agricultures third party api to search foods and meals the user has eaten and give all the nutritional value included. This application was built with JavaScript, jQuery, Ruby on Rails and Handlebars.

Survey Says! Application

Survey Says is a web app that users can make and answer survey questions. A user can sign in, create a new survey for other users to answer and they can also answer different questions presented to them. Future improvments include displaying a graph to show the percentage of users agreeing or disagreeing, also restricting users from answering more than once. Application was built using JavaScript, Handlebars, jQuery, Mongoose, Express and MongoDB.

  • Site: Survey Says!
  • Repo: Front End GitHub
  • Repo: Back End GitHub

Will Travel For Food

Will Travel for Food is a travel itinerary that a user can use to keep track of all the restaurants they would like to visit on a vacation trip. A user can make a trip with the name of the city they are going to and the date they will be arriving, then add restaurants they find using Yelp's third party api search feature included in the app. Future improvements to the app include have the user store the search results in their trips instead of filling out a form. Application was built using React and Ruby on Rails.
